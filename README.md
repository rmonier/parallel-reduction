# Parallel Reduction

## Prerequisites

- Install [Python 3.10](https://www.python.org/downloads/release/python-3109/)
- Install [Pipenv](https://pipenv.pypa.io/): `pip install pipenv`

## Installation

Install the dependencies by doing:
```sh
pipenv install --dev
```

## Usage

Run a script by doing:
```sh
pipenv run reduct
```
